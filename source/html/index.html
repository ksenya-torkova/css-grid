Список страниц

Все страницы свёрстаны под десктоп, без адаптива, зафиксирована минимальная ширина.

Страницы тестировались только в Chrome 96 на Ububntu 20.

  1. Performance - case-1.html

    Для создания сетки используются гриды с фиксированной шириной колонки и не фиксируется высота строк для корректного отображения при переполнении

  2. Traveler's blog - case-2.html

    Для создания сетки используется многомерный грид в 3 колонки и 5 строк, высота задаётся только изображениям, а у текстовых блоков - не фиксируется. Блокам задаётся индивидуальное расположение внутри грида с помощью указания нумерованных грид-линий

  3. Floral Boutique - case-3.html

    Для создания сетки каталога используется грид с размерами в относительных еденицах по ширине и автоматической высотой

  4. Furly - case-4.html

    Для создания сетки используется 4-колоночная раскладка основного содержимого и футера на гридах, а также 3-колоночный грид для хедера

  5. Let's Speak - case-5.html

    Сетка страницы построена на гридах с использованием именованых грид-областей

  6. Alt Home - case-6.html

    Сетка страницы построена на гридах с использованием многослойности
